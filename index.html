Welcome to SciView

SciView consists of two independent shiny modules for data deposition and data visualization. Browse individual application by clicking the links below! 

1. SciView: Data Visualization

The application visualises single cell expression data for studies processed through the Data Deposition module. Visualization includes interactive browsing of UMAP plots, gene expression plots, co-expression, correlations, and gene-set enrichments. Moreover, for selected genes, users can qualitatively assess expression across available studies or assess genes as potential cell-type and disease markers. It is possible to use the code as a resource by listing all single cell studies inside the helper file.

When SciView: Data Visualization is properly configured, you'll see a Shiny app in the window above. 

2. SciView: Data Deposition

The application provides an interactive form for processing single cell data. The processed database file can then be visualized using Data Visualization module. This module can be run interactively as well as programatically for easy and batch processing of single cell studies. Current set-up allows interactive data processing. While, advanced users can run the application code separately.

When SciView: Data Deposition is properly configured, you'll see a Shiny app in the window above.

Advanced users: For programatic data processing, setting up the tool as a resource, please contact the developers.
